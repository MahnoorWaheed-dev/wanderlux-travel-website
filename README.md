# 🌍 WanderLux - Travel Landing Page

✨ Explore the world like never before with **WanderLux** — a modern, visually stunning travel landing page designed to inspire adventure and showcase destinations beautifully.

## 🌐 Live Demo

👉  https://mahnoorwaheed-dev.github.io/wanderlux-travel-website/
---

## 📌 About The Project

WanderLux is a **single-page travel website** built for portfolio and practice purposes.  
It focuses on **modern UI design, smooth user experience, and aesthetic visuals** inspired by real-world travel platforms.

---

## ✨ Features

- 🌗 Light & Dark Mode toggle
- 📱 Fully Responsive Design (Mobile, Tablet, Desktop)
- 🎬 Hero Section with background image & overlay
- 🌍 Popular Destinations cards
- 🧭 Adventure Experiences section
- 💬 Traveler Stories (Testimonials)
- 🖼️ Travel Gallery grid layout
- ⚡ Smooth Animations (AOS / Custom)
- ⏳ Loading Screen Animation

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript
